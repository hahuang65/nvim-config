Section 9

Implementation Notes

Backward Compatibility

catppuccin default_integrations = false means any future plugin additions must be manually added to the integrations list. This is a maintenance trade-off for ~15ms startup savings.

Critical Warning

After changing catppuccin integrations, you must clear the compile cache: rm -rf ~/.cache/nvim/catppuccin/. Stale cache will show wrong highlights.

Performance Note

VeryLazy plugins (claudecode, amp, opencode, direnv) still contribute to total load time but fire after the --- NVIM STARTED --- measurement point, improving perceived startup.
